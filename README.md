# vue-google-map

Sample vue web application to create google maps with [vue2-google-maps]() plug-in

```
yarn add vue2-google-maps
```

## Setup

create `.env` file for configuration

```
VUE_APP_GMAP_KEY=<YOUR_GOOGLE_MAPS_API_KEY>
```
